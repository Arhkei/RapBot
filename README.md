# Rap Bot

Uses direct input to send keystrokes to game to win rap battles

## Requirements

Use the package manager [pip](https://pip.pypa.io/en/stable/)

```bash
pip install -r requirements.txt
```

## Usage
Put raps in rap bank for the bot to choose from
